# `009` Reverse Vector

## 📝 Instrucciones:

1. Crea un vector cuyos valores sean los números del 0 al 9.

2. Invierte el orden de los valores del vector e imprimelo en la consola.

## Resultado Esperado:

```bash
[9 8 7 6 5 4 3 2 1 0]
```

## 💡 Pista:

+ Así es como se obtienen los valores invertidos de un vector `array[::-1]`.
